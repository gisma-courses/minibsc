Hiking- und Mountainbike

Zwar haben Sie bereits eine Vielzahl von interessanten Aspekten für die Planung der OuterRaro-Marke vorliegen, dennoch fehlen zwei wichtige Aspekte; Sie benötigen sowohl attraktive und möglichst anspruchsvolle Mountainbike-Strecken. Für die Frage, welche Eigenschaften die gesuchten Strecken aufweisen müssen, sollten Überlegungen hinsichtlich Etappenlänge, unterschiedliche Steigungen für ein sportlich differenziertes Zielpublikum berücksichtig werden.

Doch wie können Sie die damit verbundenen Parameter aus den vorliegenden räumlichen Daten extrahieren? Letztendlich suchen ja Sie nach einer Vorauswahl von attraktiven Routen (Aussicht) nach Steigungen. Diese können auch als Widerstände oder Kosten betrachtet werden. MAn kann diese räumlichen Ausprägungen von Attraktivität und Widerständen welche über eine Distanz) eine Bilanz von "leicht oder beschwerlich", "billig oder teuer", "attraktiv oder unattraktiv" darsatellt auffassen..

Bearbeiten Sie...

  • Übersetzen Sie die gewünschten Anforderungen an die beiden Streckentypen a) Wanderweg und b) Mountainbike in konkrete Variablen bzw. Werte.
  • Suchen nach geeigneten Werkzeugen, mit welchem Sie Distanzen in Abhängigkeit von Kosten errechnen können (Tipp: Suchen Sie nach Kosten und Distanzen).
  • Identifizieren Sie einen Wanderweg, der die Insel durchquert und gleichzeitig möglichst viele Höhenmeter aufweist.
  • Identifizieren Sie eine Mountainbike-Strecke, die flache Teilstücke zur Erwärmung bzw. Entspannung und anspruchsvolle aber fahrbare Steigungen aufweist.
