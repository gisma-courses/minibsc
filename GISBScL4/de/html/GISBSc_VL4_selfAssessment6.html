Wissens-Check

Versetzen Sie sich für diese Aufgabe in die Rolle eines/r WissenschaftlerIn mit der Aufgabe, eine Multikriterien-Analyse zur Bestimmung möglicher Lebensräume des Wolfes in der Gemeinde St. Gittal durchzuführen. Ihre Aufgabe ist es, aus verschiedenen Quellen die Kriterien zu bestimmen, die die gesuchten Räume charakterisieren. Studieren Sie dazu die zwei angegebenen Texte:

  1. KORA: Dokumentation Wolf (KORA 2005)
  2. Facts: „Die Räuber kommen wieder“ (Baumgartner 1995)

Aufgabe:

Formulieren Sie mindestens drei Kriterien zur Charakterisierung potenzieller Wolfshabitate für die Gemeinde St. Gittal. Jedes Ihrer Kriterien sollte in einer räumlichen Datenebene repräsentiert werden können. Aus allen Layern soll anschließend die Gesamteignung errechnet werden. Woher Sie die Daten bekommen können, soll Sie im Rahmen dieser Übung nicht interessieren.

Addition vs. Multiplikation

Angenommen, die beiden Themenebenen aus der Abbildung Weighted Overlay im Raster- bzw. Vektormodell werden nicht addiert, sondern multipliziert. Welche wesentliche Änderung ergibt sich dann für die Resultatebene? Beurteilen Sie die folgende Aussage: „Zellen mit 0 in der Eingangsebene erhalten einen kleineren Einfluss, weil sie im Gegensatz zur Addition bei der Multiplikation in die resultierende Eignungsebene übertragen werden.“

  1. + weil + (d. h. erster Satzteil richtig, zweiter Satzteil richtig, logische Verknüpfung richtig)
  2. + / + (d. h. erster Satzteil richtig, zweiter Satzteil richtig, logische Verknüpfung falsch)
  3. + / - (d. h. erster Satzteil richtig, zweiter Satzteil falsch, logische Verknüpfung falsch)
  4. - / + (d. h. erster Satzteil falsch, zweiter Satzteil richtig, logische Verknüpfung falsch)
  5. - / - (d. h. erster Satzteil falsch, zweiter Satzteil falsch, logische Verknüpfung falsch)

Lösung (Klicken Sie hier für mehr Informationen)

4. Richtig ist, dass Zellen mit einer 0 bei der Multiplikation einen höheren Einfluss erhalten, weil (b) gilt.
