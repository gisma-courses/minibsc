The sourcecode for both firedocs and the eLML add-ons is stored on a University of Zurich server. If you want to contribute to the Firedocs project and commit source code to the project please contact us at elml@id.uzh.ch.

If you just want to have a look at the source code you can download the latest firedocs release and unzip it. Maybe you will need to replace the ending .xpi with .zip else your unzip software will not recognize the zip archive.

By the way: Once installed the Firedocs eLML Editor checks automatically if a new version is available. You can also check manually for updates in Firefox. Go to the Add-ons menu and click the "Update" button to see if new versions of one of your Firefox Add-ons are available.
